# C++ Academic Projects Portfolio

This directory contains a collection of C++ programs developed for academic purposes. Each project demonstrates core programming concepts and mathematical logic.

## 1. Square Matrix Analysis
**File:** `Matrix_Diagonal_Operations.cpp`

**Description:** This program is an academic tool designed to demonstrate matrix indexing and traversal logic. It processes a square matrix to perform specific algebraic operations:
* Identifying the **Main Diagonal**.
* Calculating the sum of the **Secondary Diagonal**.
* Computing the product of elements in the **Lower-Secondary region** (using conditional indexing $i+j > n-1$).

**Key Concepts:** Nested loops, 2D Arrays (Vectors), and Conditional Logic.
